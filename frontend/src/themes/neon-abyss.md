# Neon Abyss

既存UIの「深いネイビー背景 + シアン発光」を維持しつつ、B案（High Contrast Neon）として再調整したカスタムテーマです。

## Color Palette

- **Void Indigo**: `#090916` - 背景の基底色
- **Midnight Violet**: `#161633` - 背景グラデーション中央
- **Neon Cyan**: `#2ff6e3` - 主要アクセント
- **Neo Teal**: `#14c9b8` - アクセント濃色
- **Electric Blue**: `#6eb5ff` - 情報表示
- **Hot Pink Red**: `#ff5f88` - エラー・危険表示

## Typography

- **Headers**: Segoe UI Semibold
- **Body Text**: Segoe UI / system sans-serif

## Best Used For

暗色背景を前提に、視認性と存在感を優先したい操作パネル、録画・配信系UI、ゲーム向け管理画面。

## 根拠（現状配色）

- 選定案: `theme-ab-sample.html` の B案（High Contrast Neon）
- 主軸: シアン系を中核にし、情報/成功は寒色・警告/危険は暖色で意味分離
- 暫定判断: 現行のUI密度でもコントラストを保ちやすい配色バランス

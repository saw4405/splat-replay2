# Splat Replay ユーザーガイド

この文書は、配布版の `SplatReplay.zip` を展開して使う方向けの簡易ガイドです。
初回セットアップの細かい手順はアプリ内のセットアップウィザードに表示されるため、この文書では全体像と日常利用で迷いやすい点だけを扱います。

ソースコードからのビルドや開発環境構築は対象外です。

## 用意するもの

基本的な利用に必要なものは次の通りです。

- Windows 11 の PC
- Nintendo Switch とドック
- HDMI 入力対応のキャプチャーボード
- 安定したインターネット接続
- YouTube アカウント

使いたい機能に応じて、次も用意します。

- マイクと Groq API キー（文字起こしを使う場合）

## 使い始めるまで

1. [GitHub Releases](https://github.com/saw4405/splat-replay2/releases) から最新の `SplatReplay.zip` をダウンロードします。
2. ZIP ファイルを、自分が書き込みできるフォルダへ展開します。
3. 展開したフォルダの `SplatReplay.exe` を起動します。
4. 初回起動時はセットアップウィザードが開くため、画面の案内に従って完了します。
5. セットアップが終わると、通常の利用画面に進みます。

ウィザード内で分からない項目がある場合は、画面上で未完了になっている項目を優先して確認してください。
任意機能を使わない場合を除き、未完了のまま進めないでください。

## 使い方

### 起動する

1. Switch をキャプチャーボード経由で PC に接続します。
2. `SplatReplay.exe` を起動します。
3. メイン画面のプレビューにゲーム映像が表示されることを確認します。

### 普段の流れ

- アプリ起動後は、ゲーム映像の監視、録画、編集、アップロードが自動で進みます。
- YouTube 認証やアクセス許可が必要なときはブラウザが開くため、画面表示に従って認証を完了してください。
- 文字起こしを有効にしている場合は、録画中の音声も字幕用に扱います。

### 停止する

- 通常はアプリケーションウィンドウを閉じて終了します。
- 録画・編集・アップロード中は、処理が終わるまで閉じない方が安全です。
- 処理状況によっては、終了まで少し時間がかかる場合があります。

### スマホやタブレットから開く

同じ家庭内 LAN に接続しているスマホやタブレットから、PC と同じ画面を開けます。
無認証で公開されるため、信頼できる家庭内ネットワークでだけ使ってください。

1. PC とスマホ、またはタブレットを同じ Wi-Fi に接続します。
2. Windows のネットワーク設定で、接続中の Wi-Fi を「プライベートネットワーク」にします。
3. 管理者権限の PowerShell を開き、次のコマンドで Windows Firewall の受信許可を追加します。

```powershell
New-NetFirewallRule `
  -DisplayName "Allow TCP 5173 Inbound from LocalSubnet" `
  -Direction Inbound `
  -Action Allow `
  -Protocol TCP `
  -LocalPort 5173 `
  -RemoteAddress LocalSubnet
```

4. Splat Replay の設定画面を開き、「LAN 公開」を ON にして保存します。
5. `SplatReplay.exe` を再起動します。
6. 設定画面に表示される LAN 用 URL を、スマホやタブレットのブラウザで開きます。

## 困ったとき

### セットアップウィザードが完了できない

- 画面上で未完了になっている項目を確認してください。
- 任意機能を使わない場合は、その機能を使わない設定にして進めてください。
- 外部ツールや認証情報の準備が必要な項目は、ウィザード内の案内に従ってください。

### 録画やプレビューが始まらない

- Switch、キャプチャーボード、PC の接続を確認してください。
- OBS Studio 側にゲーム映像が出ているか確認してください。
- OBS 側で映像が出ているのに Splat Replay 側へ反映されない場合は、セットアップ時に確認した録画関連設定を見直してください。

### YouTube 認証が求められる

- ブラウザが開いた場合は、画面の案内に従って認証とアクセス許可を完了してください。
- 認証に使う Google アカウントが、セットアップ時に想定した YouTube アカウントと同じか確認してください。

### Switch の電源 OFF を検知できない

- キャプチャーボードによっては、Switch 電源 OFF 時の画面が異なります。
- 検知が安定しない場合は、電源 OFF 判定用画像の差し替えが必要になることがあります。

## カスタマイズ

### サムネイル画像を差し替える

自動生成サムネイルの上に重ねる画像を変更できます。

1. 1920x1080 の PNG 画像を用意します。
2. 展開したフォルダの `assets/thumbnail/thumbnail_overlay.png` として保存します。

透過部分がある画像は、自動生成サムネイルの上に重ねて表示されます。
完全に不透明な画像を置いた場合は、自動生成部分が見えなくなります。
